# 🎮 Game On Studio

Game On Studio is an online game development academy website built using Flask, MySQL, HTML, CSS, and JavaScript.

## Features

* Home Page
* About Us Section
* Unity Course
* Unreal Engine Course
* Contact Form
* Email Notifications
* MySQL Database
* Responsive Design

## Technologies Used

* Python
* Flask
* MySQL
* HTML
* CSS
* JavaScript

## Contact Form

When a user submits the form:

* Details are stored in MySQL.
* An email notification is sent to the website owner.
